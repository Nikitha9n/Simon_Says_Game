# Simon_Says_Game

# Live Demo:
https://simonsayzgame.netlify.app/

A simple and interactive Simon Says memory game built using HTML, CSS, and JavaScript.

## Features
- Random color sequence generation
- Level progression system
- Flash animations and game over effect
- Keyboard and button event handling

## Technologies Used
- HTML
- CSS
- JavaScript

## What I Learned
- DOM manipulation
- Event handling
- Arrays and game logic
- `setTimeout()` animations
- Random sequence generation
